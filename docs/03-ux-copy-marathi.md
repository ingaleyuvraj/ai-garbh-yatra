## 1. Onboarding

### 1.1 Splash
- App name: **गर्भयात्रा**
- Tagline: **तुमच्या गर्भारपणाची शांत सोबत**

### 1.2 Language
- Title: **तुमची भाषा निवडा**
- Marathi option: **मराठी** (default, selected)
- Helper: **तुम्ही नंतर सेटिंग्जमध्ये भाषा बदलू शकता.**

### 1.3 Intro slides
- **Slide 1** — शीर्षक: **शांत आणि सकारात्मक प्रवास**
  उपशीर्षक: **रोजची प्रेरणा, ध्यान आणि सुखद संगीतासह तुमचं मन शांत ठेवा.**
- **Slide 2** — शीर्षक: **बाळाशी प्रेमाचा संवाद**
  उपशीर्षक: **गर्भसंवाद आणि मंत्रांद्वारे तुमच्या बाळाशी जोडलं राहा.**
- **Slide 3** — शीर्षक: **तुमची रोजची सोबत**
  उपशीर्षक: **पाणी, मन:स्थिती, नोंदी आणि सौम्य दिनचर्या — सगळं एका ठिकाणी.**
- CTA: **सुरू करूया**

### 1.4 Name
- Title: **तुमचं नाव काय आहे?**
- Input placeholder: **तुमचं नाव लिहा**
- Helper: **आम्ही तुम्हाला प्रेमाने हाक मारू शकतो म्हणून.**
- Skip: **नंतर सांगेन**

### 1.5 Stage selection
- Title: **तुम्ही सध्या कोणत्या टप्प्यावर आहात?**
- Options:
  - **गर्भधारणेचे नियोजन** — *मी / आम्ही बाळासाठी तयारी करत आहोत*
  - **पहिली तिमाही** — *१ ते १३ आठवडे*
  - **दुसरी तिमाही** — *१४ ते २७ आठवडे*
  - **तिसरी तिमाही** — *२८ आठवड्यांपुढे*
- Helper: **यानुसार आम्ही तुमच्यासाठी योग्य मार्गदर्शन देऊ.**

### 1.6 Due date / week
- Title: **तुमची अंदाजे देय तारीख माहीत आहे का?**
- Option A: date picker — **देय तारीख निवडा**
- Option B: **मला आठवडा माहीत आहे** → number input **सध्याचा आठवडा**
- Option C: **मला खात्री नाही** → helper: **हरकत नाही. तुमच्या डॉक्टरांकडून ती जाणून घ्या; तोपर्यंत आम्ही सर्वसाधारण मार्गदर्शन देऊ.**

### 1.7 Partner link
- Title: **तुमच्या जोडीदाराला सहभागी करायचं आहे का?**
- Body: **जोडीदार बाळाशी संवाद साधू शकतो आणि तुम्हाला आधार देऊ शकतो.**
- CTA primary: **जोडीदार जोडा**
- CTA secondary: **नंतर करेन**

### 1.8 Notification permission
- Title: **सौम्य आठवणींसाठी परवानगी द्या**
- Body: **पाणी पिण्याची, ध्यानाची आणि गर्भसंवादाची सौम्य आठवण आम्ही करून देऊ. तुम्ही कधीही बंद करू शकता.**
- CTA: **परवानगी द्या** · **आता नको**

### 1.9 Consent + disclaimer (mandatory)
- Title: **सुरू करण्यापूर्वी एक महत्त्वाची गोष्ट**
- Body:
  > **हे app वैद्यकीय सल्ला किंवा निदानासाठी नाही.** हे केवळ तुमच्या मनःशांती, सकारात्मकता आणि दैनंदिन सोबतीसाठी आहे. कोणत्याही आरोग्यविषयक लक्षणांसाठी किंवा निर्णयासाठी कृपया तुमच्या डॉक्टरांचा सल्ला घ्या.
- Checkbox: **मी हे समजून घेतलं आहे आणि सहमत आहे.**
- Checkbox: **मी १८ वर्षांवरील आहे.**
- Links: **गोपनीयता धोरण** · **वापराच्या अटी**
- CTA (enabled after both checks): **सहमत आहे, पुढे चला**

### 1.10 Ready

- Body: **चला, तुमचा आजचा दिवस पाहूया.**
